# TechStore

E-commerce de productos de tecnología construido con Python/FastAPI en el backend,
React/Vite/Tailwind en el frontend, PostgreSQL + SQLAlchemy 2.0 (async) + Alembic
para la base de datos, autenticación JWT y pagos vía MercadoPago (sandbox).

**Estado actual:** Fase 1 (cimientos) — estructura base del backend, autenticación
JWT (register/login/me) y configuración de Alembic para migraciones asíncronas.
